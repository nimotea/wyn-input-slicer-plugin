# 简单搜索框

## install

```node
  npm i -f --legacy-peer-deps
```
**awesomes-typescript-loader依赖与typescript版本冲突，强制使用指定的ts版本**

## dev
```node
  npm run deve

  OR

  wyn-visual-tools develop
```

## package

```node
  npm run build
```

## 使用

本插件只支持搜索单一字段，如果有多字段搜索需求，可以新建计算列将不同字段关联为一个新字段，使用新字段来进行模糊搜索

**本插件类型为切片器插件并非可视化插件**
